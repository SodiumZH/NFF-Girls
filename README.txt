*NFF: Girls* mod is an taming addon of Hostile Mobs and Girls (https://www.curseforge.com/minecraft/mc-mods/hostile-mobs-and-girls).

This mod allows you to convert the hostile monster girl mobs to friendly ones, and provides multiple interaction features for them.

(This project is in development and doesn't have stable releases yet.)

Main page: https://www.curseforge.com/minecraft/mc-mods/befriendmobs-days-with-monster-girls

This mod is based on framework *NFF Services*: https://github.com/SodiumZH/NFF-Services

For tutorial, see wiki of this repository. If the wiki isn't up-to-date, see wiki/gameplay-en.md file.
